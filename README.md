# Blog FrontEnd Site
## Features
- Next.js
- Redux
- Decorator Syntax
- PostCSS

## 主要目录划分
```
.
├── actions         存放 action
├── components      存放公共组件
├── config          存放配置文件
├── pages           存放页面路由
├── reducers        存放 reducer
├── static          存放静态资源文件
├── store           存放 store
├── utils           存放工具方法
```
